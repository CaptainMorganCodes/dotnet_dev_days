## 🎯 What's Inside?

### The Game
**How it works:**
1. Open your bingo card with 25 fun questions
2. Walk around and find people who match (e.g., "Has a pet cat", "Learned Blazor this year")
3. Mark squares as you find matches
4. Get **5 in a row** to win! 🏆

It's like speed dating meets trivia – pure networking joy. Perfect icebreaker for events! 🌟

### The Tech Stack
- ⚙️ **Blazor WebAssembly** – Modern, interactive web UI
- 🏗️ **.NET 10** – Latest & greatest from Microsoft
- 🎨 **Custom CSS** – No framework bloat, just elegant styling
- 🔄 **State Management** – Real-time game logic and win detection
- 📱 **Responsive Design** – Works on desktop, tablet, and mobile

### 🤖 The Learning Path
This is more than a game – it's a **hands-on workshop** where you'll build SocOps from scratch while mastering:
- **GitHub Copilot** – AI-assisted development
- **Blazor WebAssembly** – Modern .NET web apps
- **Custom AI Agents** – Specialized Copilot skills
- **Component Design** – Building elegant UIs
- **State Management** – Real-time game logic

## 🛠️ Development

### Prerequisites
- [.NET 10 SDK](https://dotnet.microsoft.com/download/dotnet/10.0) or higher

### Available Commands
```bash
# Run the game locally
cd SocOps && dotnet run

# Build the project
dotnet build SocOps/SocOps.csproj

# Clean build artifacts
dotnet clean SocOps/SocOps.csproj
```

### Deploy
Commits to `main` automatically deploy to [GitHub Pages](https://CaptainMorganCodes.github.io/dotnet_dev_days) ✨

### Cloud Development
**GitHub Codespaces** – No local setup needed:

1. Fork this repo
2. Click **Code** → **Codespaces** → **Create codespace on main**
3. Wait for the dev container to set up
4. Run:
   ```bash
   cd SocOps && dotnet run
   ```
